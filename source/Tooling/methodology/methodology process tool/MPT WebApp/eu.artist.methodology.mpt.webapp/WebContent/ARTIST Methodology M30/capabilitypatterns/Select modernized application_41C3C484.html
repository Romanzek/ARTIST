Task Descriptor: GDEPL.A1
The modeller browses his/her workspace in order to select the modernized application for which he/she wants to generate its deployment artefacts (i.e. deployment descriptors, scripts and units). This modernised application should include all compiled code, configuration files and other artefacts required to run the application.
