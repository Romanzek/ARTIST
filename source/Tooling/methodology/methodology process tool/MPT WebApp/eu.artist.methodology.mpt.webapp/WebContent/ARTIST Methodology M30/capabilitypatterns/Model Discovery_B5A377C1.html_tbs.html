Activity: MODELDISCO
Generate the minimum set of required “raw” (i.e., low-level) initial models out of (some of) the non-cloud artefacts composing the system. The strictly required set of models actually depends on the targeted scenario and platform, but generally it includes at least models of the source code and of corresponding configuration or deployment files (e.g. in XML). This task notably implies analysing the different available non-cloud artefacts to identify the ones which are actually relevant in a given context (i.e., according to the considered migration scenario).
