Task Descriptor: EVOCOLLECT.A2
This task is focused on developing software in a task centred way. This means that the different development task and the features to be developed are managed in a bug tracking software. The developer chooses on task to work on and when she is finished she commits the code into a source code management system and references the task in the commit message. If this style of development is followed, changes of source code files and artefacts can be linked to the corresponding development tasks. For every development task the performed changes are recorded. In the opposite direction can all changes in a single file be associated with the corresponding development tasks that motivated the changes. The task centred approach can be implemented by developer discipline or can be supported by tools of the development environment.
