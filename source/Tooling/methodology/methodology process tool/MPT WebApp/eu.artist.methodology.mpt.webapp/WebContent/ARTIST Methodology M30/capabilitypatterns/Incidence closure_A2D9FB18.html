Task Descriptor: MCRP.A6
The objective of this activity is to close the incidence and assure that all the information about it is correctly collected in the incidence log.

Main Description
There are three main actions:
- Closure categorization: Check and confirm that the initial incidence categorization was correct or, where the categorization subsequently turned out to be incorrect, update the record so that a correct closure categorization is recorded for the incidence
- User satisfaction survey: Carry out a user satisfaction survey for the agreed percentage of incidences
- Incidence documentation: Chase any outstanding details and ensure that the Incidence log is fully documented so that a full historic record at a sufficient level of detail is complete.
