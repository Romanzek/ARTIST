Task Descriptor: TECHFEAS.A2
Identify migration strategy on each component. This tasks have to cover different possible migration strategy such as:
- Installation and configuration tasks: examples and complexity depend on the target Cloud provider, whether this is an IaaS or a PaaS.
- Database migration strategy, comprising data schemas, data population and querying migration, depending on compatible non-cloud and cloud data bases.
- Software migration, including adaptation to new Cloud programming paradigms.
- Components connectivity migration.
