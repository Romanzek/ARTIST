Task Descriptor: GSCODE.A2
The Code generation ToolBox (CGT) inspects the optimized PSM, its associated requirements and its associated target Cloud environment and determines the required specific target generation patterns (available in the Repository) express as transformations. Then, these patterns are applied in a transformation chain. As a result, a target PSM is obtained. This step introduces in the optimized PSM the specific target environment technologies patterns (Java, C#) required for code generation
