Task: VERBETC
Verification of the Behavioral Equivalence with Test Cases
Disciplines: Post-migration Tasks
Relationships
RolesPrimary Performer:
  • Tester
Additional Performers:
InputsMandatory:
  • Annotated PIMs
  • Existing legacy application test cases
    		Optional:
    • None
    Outputs
    • Functional test cases for the migrated application and their verdict
      Main Description
      In a first stage, existing test cases are migrated from the legacy system with the help of model abstraction and model transformations. If no or not sufficient test cases are present in the legacy application, further test cases must be derived to verify the behaviour of the migrated system by applying model-based testing techniques on an abstract representation of the system, i.e., the platform-independent model (PIM).
