      Main Description
      The Service based Software providers (SbSp) certification model checks the application against a set of best practices and standards (e.g., ISO20000 or ISO27000) considering the three aspects, finance, process and technology. A third independent party evaluates the practices of a company providing a cloud application, in order to assure by a label that the system fulfils a specified level (Gold, Silver or Bronze).
