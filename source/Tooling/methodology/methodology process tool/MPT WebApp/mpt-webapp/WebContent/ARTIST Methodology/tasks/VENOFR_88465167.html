    • Values for quality metrics in comparison with migration goals and their verdict
      Main Description
      By computing well-established quality metrics for the legacy system as well as the migrated system, we verify whether the migration goals provided by the user have been fulfilled or not. In the first phase, the computation of the metrics will take place on model-level using simulation techniques. After a satisfactory result was yielded, a more accurate computation of the metrics is done on the code-level.
