Task: VERBEUBT
Verification of the behavioural equivalence with end user-based tests

      • Real-life functional test cases and their verdict
        Main Description
        Interaction sequences of a user with the migrated system are recorded and equivalent interactions are executed on the legacy system. Using abstraction, we test whether both sequences triggered equivalent behaviour in both systems, and whether they produce the same output. To perform these tests, both systems must run in parallel and real users or respective log files are needed for generating the test cases, i.e., the interactions.
