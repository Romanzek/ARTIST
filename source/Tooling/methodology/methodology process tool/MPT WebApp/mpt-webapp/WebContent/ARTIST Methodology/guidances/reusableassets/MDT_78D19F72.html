Main Description

‚óè     Model Discovery Tool (MDT) enables modelers to obtain raw (e.g. low level) platform specific models (PSM) that describe the legacy application and its artifacts. MDT uses a taxonomy of legacy artifacts in order to guide the detection, classification and parsing of such artifacts and incorporate their relevant content to the raw PSM. ARTIST offers two specializations of MDT: Java Discovery Tool and .NET Discovery Tool, specialised on the analysis and modelling of Java and .NET legacy application, respectively.
