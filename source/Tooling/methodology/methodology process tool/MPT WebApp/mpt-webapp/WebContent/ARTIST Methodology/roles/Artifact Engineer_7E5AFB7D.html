Role: Artifact Engineer
Role Sets: Migration Artefacts Reuse And Evolution Roles
Relationships
Modifies
  • A list of potentially reusable artefacts to be published to the ARTIST marketplace
  • An additional list of changes to be captured
  • Artefacts are inserted in the publishing queue of the marketplace
  • Extended set of changes to move the artefact from one consistent state to another
  • Formal listing of the changes to be made to the artefact in terms of primitive and complex changes
  • List of incosistencies (issues) to be resolved
  • The new version of the artefact in a consistent state
    Main Description
    An artefact engineer is someone who specifies an artefact (meta-models, M2M transformations, M2T transformations, etc.) to be committed in the repository. Furthermore, this person is involved in maintaining the artefacts throughout its lifetime.
