      Outputs
      • Low-level models
        Main Description
        Generate the minimum set of required “raw” initial models out of (some of) the legacy artefacts composing the system. This task notably implies analysing the different available legacy artefacts to identify the ones which are actually relevant in a given context (i.e.; according to the considered migration scenario).
        Steps
        S1: Analyse and classify the existing legacy software artefacts to be migrated according to the taxonomy
        S2: Perform the actual model discovery
