ARTIST Repository server runtime
================================

Content:
    - repo.server.war
      The web archive containing the ARTIST repository server
    - apache-tomee-runtime
      The Apache TomEE 1.6 server runtime with the 
      Modeshape JCR database and the ARTIST repository server
      application pre-installed. 

To start the repository server, extract the zip archive, 
go to the "apache-tomee-runtime/bin" folder 
and execute 

    "startup.bat" or "startup.sh" 

depending on your environment. The server will run on port 8081.

The ARTIST repository server is released under EPL v1 license.
